## ✨ Features

* **Dynamic Rendering:** Questions and options are generated dynamically via JavaScript from a structured JSON dataset.
* **Instant Feedback:** Validates user input immediately, highlighting the selected option in green (correct) or red (incorrect).
* **State Management:** Utilizes the browser's `LocalStorage` API to cache session progress, allowing users to safely refresh or return later without losing their score or current question.
* **Multiple Quiz Modes:** * Week-by-week selection (Weeks 1-12)
  * Full syllabus mock test
  * Randomized 20-question generator
* **UI/UX Design:** Includes a system-agnostic Dark/Light mode toggle, an interactive progress bar, and a responsive layout for mobile and desktop screens.
* **Post-Quiz Review:** Generates a comprehensive score report detailing correct/incorrect answers alongside educational explanations.
